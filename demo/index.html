RingCentral JS Widgets

Below code snippet will produce a functionality-ready RingCentral WebPhone 

<rc-phone 
    first-level=dial-pad,time-line 
    width=250
    height=400
    key=YOUR_APP_KEY
    secret=YOUR_APP_SECRET
    fixed>
</rc-phone>
<script async defer id='rc-widgets-script' 
src='https://ringcentral.github.io/ringcentral-js-widget/build/host.js'></script>
The embedded web phone can be interacted with main pages, such as click-to-dial.
